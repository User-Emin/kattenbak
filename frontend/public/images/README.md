## Afbeeldingen converteren naar webp:

### Online:
- https://cloudconvert.com/jpg-to-webp
- https://www.freeconvert.com/jpg-to-webp

### Command line (als je webp tools hebt):
```bash
cwebp input.jpg -q 85 -o output.webp
```
